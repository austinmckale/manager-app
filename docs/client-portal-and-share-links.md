# Share Links (no client portal)

**This app is management software for the owners.** There is no client portal. You can still generate **share links** from a job hub to send the client a read-only timeline or gallery (e.g. by text or email).

---

## Share links (timeline or gallery, read-only)

- **Timeline link** (`/share/{token}` type TIMELINE): shared timeline view.
- **Gallery link** (`/share/{token}` type GALLERY): shared photo gallery (selected “client visible” photos).

**Where you create them:** Job hub → section **“5) Share links”** → Generate timeline link or Generate gallery link. Send the URL to the client by text or email.
